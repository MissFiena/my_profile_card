## ✨ Features
- 🖼️ Parallax hero background image
- 👤 Oval profile picture overlapping hero
- 📋 Personal info card
- 💬 Auto-rotating quote
- 📚 Course list — Past / Current / Upcoming (grouped by category)
- 📧 Contact buttons — Email, Phone, GitHub, LinkedIn
- 🔽 Expandable bio (Read More / Read Less)
